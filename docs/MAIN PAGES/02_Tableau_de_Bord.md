# Tableau de Bord

## Page Overview

Centre de performance et statistiques global. Dashboard analytique complet affichant les revenus, commandes, bénéfices, performance par wilaya, analyse financière avec calculateurs de profit/IFU, et outils de reporting avec exports et intégrations.

**Route**: `/analytics`  
**Fichier Principal**: `src/app/analytics/page.tsx`
