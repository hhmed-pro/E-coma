# Operations Zone

## Zone Overview

The **Operations Zone** (Red) focuses on "Run the Business" activities. It handles the core logistical and fulfillment workflows necessary to process orders and manage inventory.

| Feature | Description |
| :--- | :--- |
| **Focus** | Order Processing, Logistics, Recovery |
| **Color** | Red (`#EF4444`) |
| **Icon** | Phone / Truck |
